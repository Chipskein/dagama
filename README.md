# [Dagama](https://dagama.herokuapp.com/)
## Descrição
  Rede Social em PHP para disciplina de banco de dados(2021). 
## Nota
  Para utilizar o sistema de email 
  Caso rode local 
####  Use uma das alternativas abaixo:

  * localhost:8081 
  * localhost:8082
